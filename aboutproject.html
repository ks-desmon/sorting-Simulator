Information
The name of the project "Sorting Simulator" defines the meaning of the project. This project is based on the sorting algorithms that how they work, what are the logics in the sortings and how they all are different from one other. The main motive of this project is to find out which sorting is the best which takes less time to sort the data in fastest way.
Why Do we use sorting ?
"Sort" means to show the data in systematic order (descending/ascending). We may have to use it some time just because the data is not in the perfect order or not easily readable/understandable, so we use some techniques or algorithms of sorting to sort out the data in the perfect order. There are many algorithms which can be used but we have to use the best techniques among them to sort the data efficiently and fastly.
